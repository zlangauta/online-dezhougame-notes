# online-dezhougame-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容主要与在线德州游戏相关的笔记、工具或参考资料有关，但仓库本身不针对任何特定网站或服务。

## 项目简介

本项目是一个静态 HTML 页面集合，旨在以轻量、可离线查看的方式，整理和分享与在线德州游戏相关的学习笔记、策略备忘或辅助工具。所有页面均为独立文件，无需后端支持，可直接在浏览器中打开使用。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库导航页（如有）
├── notes/             # 笔记类页面目录
├── tools/             # 工具类页面目录
└── archive/           # 归档或历史版本页面目录
```

各目录下均为独立的 `.html` 文件，文件名通常能反映页面内容主题。具体文件说明请参考各页面内的注释或导航。

## 页面归档说明

- 每个 HTML 页面均为完整独立的文档，包含自身所需的样式与脚本。
- 页面可能引用外部资源（如字体、图标库），但不依赖任何特定后端服务。
- 归档内容可能随时间更新或移除，不保证所有页面的长期可用性。
- 部分页面可能包含实验性或未完成的内容，请按需使用。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交建议或报告问题，但无法保证及时处理。
- 不接受直接向仓库推送修改的 Pull Request，如有改进建议请先讨论。
- 仓库内容仅供学习参考，不构成任何形式的建议或指导。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可查看内容。
3. 也可通过 GitHub Pages 等静态托管服务在线浏览（需自行配置）。
